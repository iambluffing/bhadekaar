# BhadeKaar

Equipment, Materials & Services – All in One Place.

BhadeKaar is a Goa-based marketplace connecting customers with commercial vehicle providers, construction material suppliers, and professional service providers.

## Services

* Commercial Vehicles

  * JCB
  * Excavators
  * Tipper Trucks
  * Pickup Vehicles
  * Transport Services

* Construction Materials

  * Cement
  * Sand
  * Aggregates
  * Bricks
  * Steel

* Professional Services

  * Electricians
  * Plumbers
  * Carpenters
  * Welders
  * Technicians

## Features

* Customer Registration
* Vendor Registration
* Admin Dashboard
* Vendor Dashboard
* Customer Dashboard
* Request for Quotation (RFQ)
* Vendor Bidding System
* Commission-Based Marketplace
* Google Maps Integration (Planned)
* Online Payments (Planned)

## Service Area

Currently serving Goa, India.
